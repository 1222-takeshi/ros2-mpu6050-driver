# ros2_mpu6050_driver
This repository assumes that you are using the MPU6050 with a Raspberry Pi.

# Tested environment
I have tested it in the following environment.
- Raspberry Pi 3B+
- ubuntu 20.04
- ROS2 foxy

# requirements
- WiringPI

# Usage
``` sh
ros2 launch imu_driver mpu6050_driver.launch.xml 
```
